# Розыгрыш по RFID-метке

Статический сайт для GitHub Pages. При открытии сайта отображается клон стандартной страницы ошибки браузера, а после нажатия кнопки запускается зацикленный звук из `sound.mp3`.

## Файлы
- `index.html` — главная страница
- `styles.css` — стили для клонов ошибок
- `script.js` — обнаружение браузера, управление историей и запуск звука
- `sound.mp3` — пользовательский звуковой файл, должен лежать рядом с сайтом

## Установка
1. Поместите `sound.mp3` в папку с сайтом.
2. Разверните сайт на GitHub Pages из ветки `main` или `gh-pages`.

## Особенности
- Клонирует вид ошибки для Chrome, Safari, Firefox, Edge и Яндекс.Браузера
- Старт звука происходит только после клика по кнопке
- Звук зациклен и скрыт от пользователя
- Браузерная кнопка «Назад» перенаправляет на ту же страницу
- Попытки обновления через F5/Ctrl+R блокируются на стороне страницы
